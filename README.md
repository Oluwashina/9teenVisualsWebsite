# 9teen Visuals | Photography Portfolio

A premium, cinematic photography portfolio designed for 9teen Visuals. This platform showcases professional photography across multiple categories and includes an integrated administrative dashboard for real-time portfolio management.

## ‚ú® Key Features

- **Cinematic Experience**: High-end visual design with glassmorphism, smooth animations, and a focus on visual storytelling.
- **Dynamic Galleries**: Dedicated sections for **Portraits**, **Events**, and **Baby Pictures**.
- **Admin Portfolio Manager**: A private dashboard (`/#admin`) to upload, categorize, and remove photographs without touching the code.
- **Direct WhatsApp Booking**: A streamlined inquiry form that redirects clients directly to WhatsApp with pre-filled project details.
- **Responsive Design**: Optimized for a premium experience on everything from ultra-wide monitors to mobile devices.
- **Lightning Fast**: Built with Vite and Vanilla JavaScript for maximum performance and minimal footprint.

## üöÄ Getting Started

### Local Development

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Oluwashina/9teenVisualsWebsite.git
    cd 9teenVisualsWebsite
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Production build**:
    ```bash
    npm run build
    ```

## üîí Administrative Access

The **Portfolio Manager** is hidden from the public-facing navigation to maintain a clean aesthetic. To manage your visual assets, navigate to:

`YOUR_DOMAIN/#admin`

From here, you can:
- Upload new images.
- Assign categories (Portrait, Event, Baby Picture).
- Remove existing assets.

*Note: In the current version, portfolio data is persisted using internal state and browser storage.*
